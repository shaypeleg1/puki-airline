# VueJS Basic Seed

Including:
1. Webpack 
2. ES6 with babel
3. Basic routing
4. Unit Testing with Karma

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
